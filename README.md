# Dicoding Collection Dashboard ✨
Dashboard interaktif berbasis Streamlit untuk menganalisis data penyewaan sepeda menggunakan dataset Bike Sharing.
# Features
Visualisasi jumlah penyewaan sepeda harian
Analisis penyewaan berdasarkan musim
Analisis penyewaan berdasarkan tahun
Filter rentang tanggal
Insight interaktif menggunakan Streamlit
# Dataset
Dataset berisi informasi mengenai:
  - tanggal penyewaan
  - musim
  - kelembaban
  - hari kerja
  - jumlah penyewaan sepeda
# Setup Environment - VSCode
  - pip install pandas, matplotlib, seaborn, streamlit, babel
  - pip install -r requirements.txt
# Run steamlit app
streamlit run dashboard.py
